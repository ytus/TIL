## flowtype
When a class implements an interface, there is no `implements` key word. Instead, declare it like this:

~~~ js
interface ISpeak {
  speak(): string;
}

class Person {
  constructor(name) {
    (this: ISpeak)  // <--- HERE
    // ...
  }
  
  speak() {
    // ...
  }
}
~~~
